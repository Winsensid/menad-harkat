## Setup

1. Clone or download this repository
2. Add images to the `images/` folder:
   - `menad-portrait.jpg` - Portrait for About section
   - `album1.jpg`, `album2.jpg`, `album3.jpg`, `album4.jpg` - Album covers
   - `gallery1.jpg` through `gallery6.jpg` - Gallery photos
3. Open `index.html` in your browser

## Image Requirements

Place the following images in the `images/` folder:

- **menad-portrait.jpg**: A professional portrait photo (recommended: 800x1000px)
- **album1.jpg - album4.jpg**: Album/single cover art (recommended: 600x600px)
- **gallery1.jpg - gallery6.jpg**: Performance and event photos (recommended: 800x600px)

## Customization

### Colors
The color scheme uses Algerian flag colors. You can modify them in `styles.css`:
```css
:root {
    --primary-color: #006233;    /* Green */
    --secondary-color: #D52B1E;  /* Red */
    --gold-color: #D4AF37;       /* Gold */
}
```

### Content
- Edit text content in `index.html`
- Update social media links in the contact section
- Add real email/phone in the contact info

### Functionality
- Connect the contact form to a backend service (e.g., FormSpree, EmailJS)
- Link music cards to streaming platforms (Spotify, YouTube, etc.)
- Add real music player integration
